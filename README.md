# Article Database

## Technologies

* Node.js
* Express
* Express Session, Validation, Messages, Connect-flash 
* Pug Templating
* MongoDB & Mongoose
* Passport
* Bcrypt
* Nodemailer
* Multer

### Installation

Install the dependencies

```sh
$ npm install
```
Run app

```sh
$ npm start
```

And point your browser to `http://localhost:3001`.
